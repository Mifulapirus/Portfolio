# Interactive 3D Lab Experience

An immersive 3D electronics lab where you can explore projects by walking around as an animated character.

## Features

- **3D Environment**: Fully navigable electronics/robotics lab built with Three.js
- **Animated Character**: Simple character with walking animations
- **WASD/Arrow Controls**: Move freely around the lab
- **Interactive Stations**: Walk to glowing project areas to view details
- **Project Displays**: Each station shows project information in a modal
- **Smooth Camera**: Camera follows character with smooth transitions
- **Visual Feedback**: Glowing rings and floating holograms mark project areas

## Technologies Used

- **Three.js**: 3D rendering and scene management
- **ES6 Modules**: Modern JavaScript with import maps
- **HTML5 Canvas**: WebGL rendering
- **CSS3**: Styled UI overlays and modals

## Projects Featured

- PawTracker (IoT/LoRa GPS tracker)
- Noter (AI audio transcription)
- CSA (Full-stack Stïk Registry)
- Democratic-Robot (ESP32 robotics)
- Arduino Logger (C++ library)
- KiCad Components (Hardware design)

## Usage

1. Open `index.html` in a modern browser (Chrome, Firefox, Edge)
2. Use **WASD** or **Arrow Keys** to move the character
3. Walk to any glowing area to view project details
4. Character automatically looks up when near a station
5. Close modal to continue exploring

## Controls

- **W / ↑**: Move forward
- **S / ↓**: Move backward
- **A / ←**: Move left
- **D / →**: Move right

## Browser Requirements

- Modern browser with WebGL support
- ES6 module support
