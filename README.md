# Portfolio

This is a personal portfolio project showcasing various works and projects.

## Contents

- **index.html**: Main homepage with hero section, about, projects grid, and contact.
- **style.css**: Custom CSS for styling.
- **images/**: Directory for project thumbnail images (add placeholder.jpg or actual images here).

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5 for responsive design

## Usage

Open `index.html` in a web browser to view the portfolio site.

## Deployment

Host on a web server or deploy to GitHub Pages. Add project images to the `images/` folder and update the src attributes in index.html.
